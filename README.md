# Two Ways of Measuring

**A Framework for Distance, Memory, and Fault-Tolerant Computation**

An exploration of measurement frameworks contrasting Archimedean and
ultrametric distance, with implications for memory and fault-tolerant
computation -- part of the QNFO research portfolio.
